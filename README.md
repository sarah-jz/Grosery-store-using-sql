*Features*
Admin login system
Add new products to stock
Update product price
Delete products
View all available stock
Customer purchase system
Stock quantity updates automatically
Bill/receipt generation

*Technologies Used*
Python 3
MySQL Database
mysql-connector-python

Start MySQL server
Make sure MySQL is running on your system.

*How It Works*
Admin Side:
Login using password
Manage inventory (add/update/delete products)
Change admin password
View stock details

Customer Side:
View available products
Select items to purchase
System checks stock availability
Generates final receipt with total cost
